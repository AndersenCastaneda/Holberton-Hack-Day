Algorithm put in place

(screenshot)

Before run our core algorithm, we generate a random ‘Master code’ of 4 slots, every slot can be some of the 6 available colors.

Then we wait for the user to submit his selections and we run the core algorithm:

We have a loop to compare every ‘Master code’ slot with the ‘User code’ slot in the same position, if they have the same color we set the corresponding feedback (one red ping) and we update the array ‘userIsEvaluated’ wich indicates wich slots sould’t be take in account in later iterations.

If they haven’t the same color we start another loop to looking for some other slot with the same color and we set the corresponding feedback (one white ping).
